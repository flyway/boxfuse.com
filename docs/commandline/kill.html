kill

Gracefully kill an Instance or an App either on your machine or on AWS
Specifying an App will kill all Instances of that App.

Usage: boxfuse kill app | instance

> boxfuse kill vb-ec717c5e

Killing Instance vb-ec717c5e on VirtualBox ...
Instance killed in 00:03.493s

Settings

Argument boxfuse.conf property Default Description
-env=... env dev Select the specified environment (ex.: test or prod)
